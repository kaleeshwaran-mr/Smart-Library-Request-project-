# Smart-Library-Request-project-
# 📚 Smart Library Request System

## 📌 Project Overview

This project is developed as part of the **Naan Mudhalvan ServiceNow training program**.
The Smart Library Request System helps users to request books easily and allows admin to manage and track requests efficiently.

---

## 🎯 Objectives

* To simplify book request process
* To maintain digital records of library requests
* To automate approval workflow
* To improve user experience

---

## 🛠️ Technologies Used

* ServiceNow Platform
* JavaScript
* Glide System (ServiceNow APIs)

---

## ⚙️ Features

* 👤 User Login
* 📖 Book Request Submission
* 🔄 Request Approval Workflow
* 📊 Status Tracking
* 📁 Record Management

---

## 🔄 Workflow

Login → Submit Request → Admin Review → Approval/Reject → Status Update → Logout

---

## 📂 Project Files

* Business Rules
* Client Scripts
* UI Policies
* Tables and Forms
* Script Includes

---

## 🚀 How to Run

1. Login to ServiceNow instance
2. Navigate to the application
3. Submit a new request
4. Track request status
